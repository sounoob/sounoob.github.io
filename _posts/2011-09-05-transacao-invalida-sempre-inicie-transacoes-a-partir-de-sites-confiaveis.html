Tenho visto na internet muita gente reclamando que ao tentar implantar o PagSeguro no sistema é exibido o erro: "Transação inválida. Sempre inicie transações a partir de sites confiáveis."

No dia 3 de agosto de 2011 o PagSeguro lançou as famosas API, uma delas é a de pagamento, onde a finalidade é fazer uma integração mais dinâmica e ao mesmo tempo segura. O fato é que junto com o lançamento dessa API foi liberado uma opção que permite que a conta receba apenas pagamento usando essa nova API o que pode acabar gerando esse erro.

Muita coisa mudou de lá até aqui, antes você bloqueava os formulários HTML para receber só via API, mas com o fato dos formulários não receberem mais atualizações a opção está inversa. Você desbloqueia sua conta para receber formulários HTML e assim evitar mostrar esse erro.

Para desbloquear clique no link: Pagamento via Formulário HTML

Na tela que irá carregar você irá marcar a opção: "Deseja ativar Pagamentos via Formulário HTML?" e clicar em salvar.

Pronto agora você já poderá vender com o PagSeguro, mas atenção, formulários estão defasados e não são seguros, alguém pode burlar sua venda e fazer um belo de um estrago... Melhor usar as APIs, se você não for programador ou não souber programar, pague alguém...
