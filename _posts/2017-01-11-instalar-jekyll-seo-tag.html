Jekyll SEO Tag que vai te ajudar a criar metatags para otimizar o rastreio da sua página, quanto a exibição nas redes sociais.

Siga os passos abaixo:
  1. Adicione no arquivo Gemfile 
    gem 'jekyll-seo-tag'
  2. Adicione dentro da tag head 
    {% raw %}{% seo %}{% endraw %}
  3. Adicione no arquivo _config.yml 
    gems:
    - jekyll-seo-tag
    Agora precisamos colocar as configurações padrões do teu site. Utilize as seguintes configurações nesse mesmo aquivo.
    • title - O título do seu site
    • description - Uma breve descrição do site
    • url - a URL completa para a raiz do seu site
    • author - O autor padrão dos seus posts
    • logo - A URL do logo do site
    • google_site_verification - O código de verificação do Google webmaster tools
    • facebook
      • facebook:app_id - O ID do aplicativo
      • facebook:publisher - A URL da página do Facebook
      • facebook:admins - O ID de usuário do Facebook que gerencia a página
      Deverá ser algo parecido com o código abaixo. 
      facebook:
    app_id: 1234
    publisher: 1234
    admins: 1234
    • twitter:username - A conta do twitter 
      twitter:
    username

Abaixo as configurações feita no _config.yml que podem ser reescrita no YAML front matter da sua página, post ou documento.

  • title - Título da página
  • description - Uma breve descrição do conteudo da página
  • author - autor do post
  • image - URL de alguma imagem para ser associada ao post
