Tudo que é código o Jekyll tenta executar, mas as vezes queremos apenas mostrar um trexo deste código em um post ou página. Vem e veja como forçar a exbição.

Em minha jornada de blogueiro passei raiva com o Jekyll enquanto tentava escrever sobre como programar com Jekyll.

Digamos que você precise mostrar o seguinte código:

{% highlight jekyll linenos %}{% raw %}{% if post.size > 0 %} {% endif %}{% endraw %}{% endhighlight %}

Esse apenas copiando e colando ele será executado, já que o Jekyll consegue entender a instrução.

Para evitar que o Jekyll execute o código, insira-o dentro de raw, exatamente como no código abaixo:

{% highlight jekyll linenos %}{% raw %}{% raw %} {% if post.size > 0 %} {% endif %} {% {% endraw %}endraw %}{% raw %} {% endraw %}{% endhighlight %}

Bom, por hoje é só. Espero que tenham gostado da novidade.
